# VoidCare

Premium Windows x64 terminal tool for offline security checks, suspicious-file quarantine, and safe system optimization.

Developed by Ysf (Lone Wolf Developer)

## What This Tool Does

- Runs as a terminal-only Windows utility built with C# .NET 8.
- Shows system status, elevation state, Defender availability, and registered antivirus products.
- Audits startup persistence items such as Run keys, startup folders, scheduled tasks, and services.
- Uses Microsoft Defender for scans and remediation when Defender is available.
- Finds suspicious files with heuristic checks and uses a quarantine-first workflow.
- Provides safe cleanup and optimization actions with confirmation for risky operations.
- Lists installed apps, curated removable bloat candidates, and tool logs.

## Safety

- Windows x64 only
- Offline-only
- No telemetry
- Defender-only remediation
- Suspicious results are heuristic-only
- Default suspicious-file action is quarantine
